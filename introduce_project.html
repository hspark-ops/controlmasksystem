  마스크 쓰기 생활화

  호로록조가 함께 합니다.


  마스크 출입 통제 시스템(Control Mask System)

    시스템 개요

      마스크 착용 여부를 판단하여 출입을 통제 하는 시스템

    사용 기술

      Tensorflow       Tensorflow keras       Tensorflow lite

    CMS 알아보기

      1.이미지 학습 : 수십 수천장의 학습 할 이미지를 수집합니다.

      2.모델링 : 학습 할 이미지를 가지고 데이터 모델을 구성한다.   (ex. 마스크 on, 마스크 off)

      3.모델 생성 : 이미지로 학습된 데이터 모델을 생성한다.

      4.모델 주입 : 생성된 모델에 새로운 이미지를 주입한다.

      5.마스크 탐지 : 마스크 on/off 결과를 확인한다.
